### Boite aux lette connectée

# Composant
ESP8266 D1
HX711
Ballance de cuisine
LED
resistance 4,7K
Photodiode
Chargeur solaire

# Principe
Utilisation de l'energie du panneau solaire (en route depuis plus de 18 mois !)
Utilisation de la luminosité ambiante pour réduire la cadence de pris de mesure (le facteur passe rarement la nuit).
Auto tarre lorsque le poids décroit (relevée de courrier)
Post Domoticz pour le relevé du courrier (+ alerte SMS via Domoticz + Ok Google via IFTTT)
